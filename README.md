# Java Modern Banking Application

A professional, modular Java Swing banking application featuring a modern dark-themed UI, custom graphics, secure password handling, and SQLite database persistence.

## Features
* Modern dark-themed user interface
* Database connectivity (SQLite)
* Secure login and authentication
* Transaction history and balance checking
* Custom painted UI components

## How to run locally
1. Clone or download this project.
2. Open the project folder.
3. Double-click the `run.bat` file to compile and execute the application.

*Requires Java installed on your machine.*
